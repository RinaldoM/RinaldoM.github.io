What is Bootstrap?

Bootstrap is a free open source CSS framework. The framework has a lot of standard functions that make developing a responsive website very fast. In addition, Bootstrap also has many HTML, CSS and JavaScript templates.

How does it work?

Let's say you want to add a button to your website but also want to style it a bit. You can do that by writing a few lines of CSS code. But you can also use Bootstrap's CSS template.

First of all you need to create the link to the css file in your <head>

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.1.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-1BmE4kWBq78iYhFldvKuhfTAU6auU8tT94WrHftjDbrCEXSU1oBoqyl2QvZ6jIW3" crossorigin="anonymous">

Next you need to add the style class to your button in your html.

<button type="button" class="btn btn-primary">Primary</button>
